# LSTM Forecaster (LSTM 시계열 예측 모델)

> LSTM으로 슬라이딩 윈도우 입력을 받아 다음 타임스텝 값을 예측하는 지도학습 기반 시계열 예측 모델

## 실행 방법

```bash
cd timeseries-ex/forecasting
python forecasting.py
```

의존성: `torch`, `numpy`

500포인트의 합성 시계열(sin + 선형 트렌드 + 노이즈)을 정규화 후 윈도우 크기 20으로 슬라이딩하여 학습한다.

## 핵심 개념

### 슬라이딩 윈도우
시계열을 고정 크기 윈도우로 분할하여 지도학습 문제로 변환한다.

```
[t-19, t-18, ..., t-1, t] -> 예측: [t+1]
```

이 코드에서는 `window_size=20`, `pred_len=1` (다음 1스텝 예측).

### LSTM 구조
```
입력 (batch, 20, 1) -> LSTM (hidden=32) -> 마지막 타임스텝 은닉 (batch, 32) -> FC -> 예측값 (batch,)
```

- LSTM의 마지막 출력 `output[:, -1, :]`만 예측에 사용
- 셀 상태(c_n)와 은닉 상태(h_n)로 장기 의존성 포착

### 정규화
```python
series = (series - series.mean()) / series.std()
```

학습 전 z-score 정규화로 그래디언트 안정성과 수렴 속도 향상.

### 평가 지표
| 지표 | 수식 | 의미 |
|------|------|------|
| MAE | mean(|pred - actual|) | 평균 절대 오차, 이상치에 강건 |
| RMSE | sqrt(mean((pred - actual)^2)) | 큰 오차에 더 민감 |
| Direction Accuracy | (부호 일치 수) / 전체 | 상승/하락 방향 예측 정확도 |

방향 정확도는 절대값보다 방향이 중요한 금융 예측 등에서 핵심 지표이다.

### 배치 학습
```python
perm = torch.randperm(split)  # 에폭마다 셔플
for i in range(0, split, batch_size):
    idx = perm[i:i+batch_size]
```

전체 시퀀스를 배치로 나누어 SGD 노이즈로 일반화 성능을 높인다.

## 관련 모델과의 비교

| 항목 | LSTM | Transformer | ARIMA | Prophet |
|------|------|-------------|-------|---------|
| 장기 의존성 | 중간 (게이트로 완화) | 강함 (어텐션) | 약함 | 중간 |
| 비선형 패턴 | 가능 | 가능 | 불가 | 제한적 |
| 학습 데이터 | 중간 필요 | 많이 필요 | 적어도 가능 | 적어도 가능 |
| 추론 속도 | 순차적 (느림) | 병렬 (빠름) | 빠름 | 빠름 |
| 해석 가능성 | 낮음 | 어텐션 맵 | 높음 | 높음 |

### LSTM vs GRU
- **LSTM**: 셀 상태(c_t) + 은닉 상태(h_t), 입력/망각/출력 3개 게이트
- **GRU**: 은닉 상태만, 리셋/업데이트 2개 게이트, 파라미터 수 33% 적음
- 실무에서는 성능 차이가 크지 않아 GRU를 먼저 시도하는 경우가 많음

### 단일 스텝 vs 다중 스텝 예측
| 전략 | 방법 | 특징 |
|------|------|------|
| 단일 스텝 (이 코드) | pred_len=1 | 정확도 높음 |
| Direct Multi-step | 출력 크기 N | 병렬 예측, 오차 미전파 |
| Recursive | 이전 예측을 다음 입력으로 | 오차 누적, 단순 구현 |
| SEQ2SEQ | 인코더-디코더 | 유연한 출력 길이 |
