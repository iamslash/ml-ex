# Scratch 실습 (From Scratch 구현)

> 외부 프레임워크 없이 순수 Python(+ numpy)으로 핵심 알고리즘을 직접 구현한다.

## 학습 순서

| # | 모델 | 디렉토리 | 핵심 개념 |
|---|------|----------|----------|
| 1 | Linear Regression | [linear-regression](linear-regression/) | 경사하강법, MSE, 최적화 |
| 2 | Logistic Regression | [logistic-regression](logistic-regression/) | 시그모이드, 이진 분류, cross-entropy |
| 3 | Softmax Regression | [softmax-regression](softmax-regression/) | 다중 분류, softmax, one-hot |
| 4 | kNN | [knn](knn/) | 거리 기반 분류, 비모수 모델 |
| 5 | Decision Tree | [decision-tree](decision-tree/) | 정보 이득, 엔트로피, 재귀 분할 |
| 6 | MLP (Micrograd) | [micrograd](micrograd/) | autograd, backprop, chain rule |
| 7 | CNN (최소 구현) | [cnn-minimal](cnn-minimal/) | 합성곱, 풀링, 특징 추출 |
| 8 | Self-Attention | [self-attention](self-attention/) | QKV, 스케일드 닷 프로덕트 |
| 9 | GPT (MicroGPT) | [microgpt](microgpt/) | 디코더, 인과적 어텐션, 생성 |
| 10 | Q-Learning | [q-learning](q-learning/) | 보상, 상태-행동 가치 |
