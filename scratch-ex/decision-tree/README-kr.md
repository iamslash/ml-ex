# Decision Tree (결정 트리)

> 특성 값을 기준으로 데이터를 재귀적으로 분할하여 분류하는 모델.

## 실행 방법

```bash
cd scratch-ex/decision-tree
python decision_tree.py
```

## 핵심 개념

### 알고리즘

```
1. 모든 특성과 임계값 조합에 대해 정보 이득 계산
2. 정보 이득이 가장 큰 (특성, 임계값) 선택
3. 데이터를 두 그룹으로 분할
4. 각 그룹에 대해 재귀적으로 반복
5. 종료 조건: 순수 노드, 최대 깊이, 또는 이득 없음
```

### 엔트로피 (Entropy)

```
H(y) = -Σ p_k * log2(p_k)
```

- 데이터의 불순도(impurity) 측정
- 모든 데이터가 같은 클래스 → H = 0 (순수)
- 균등하게 섞여 있으면 → H 최대 (불순)

### 정보 이득 (Information Gain)

```
IG = H(parent) - [n_left/n * H(left) + n_right/n * H(right)]
```

- 분할 전후의 엔트로피 차이
- 클수록 좋은 분할

### kNN과의 비교

| | kNN | Decision Tree |
|---|---|---|
| 학습 | 없음 | 트리 구축 |
| 예측 | 전체 데이터와 거리 계산 | 트리 탐색 (빠름) |
| 해석 가능성 | 낮음 | 높음 (규칙으로 설명 가능) |
| 결정 경계 | 거리 기반 (원형) | 축 정렬 (직사각형) |

### Random Forest로의 확장

Decision Tree 여러 개를 앙상블하면 Random Forest가 된다.
- 각 트리는 데이터의 부분 집합으로 학습 (bagging)
- 각 분할에서 특성의 부분 집합만 사용
- 다수결 투표로 최종 예측
