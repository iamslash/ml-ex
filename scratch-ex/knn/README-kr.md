# k-Nearest Neighbors (kNN)

> 가장 가까운 k개의 이웃을 찾아 다수결로 분류하는 비모수 모델.

## 실행 방법

```bash
cd scratch-ex/knn
python knn.py
```

## 핵심 개념

### 알고리즘

```
1. 쿼리 점과 모든 학습 데이터 사이의 거리 계산
2. 거리가 가장 가까운 k개를 선택
3. k개의 라벨 중 다수결 투표로 분류
```

학습 단계가 없다 (lazy learning). 모든 계산은 예측 시점에 수행된다.

### 유클리드 거리

```
d(a, b) = sqrt(Σ(a_i - b_i)²)
```

### k 값의 영향

| k | 특성 |
|---|------|
| 작을 때 (1, 3) | 결정 경계가 복잡, 노이즈에 민감 (overfitting) |
| 클 때 (11, 15) | 결정 경계가 부드러움, 세밀한 패턴을 놓침 (underfitting) |

이것이 **bias-variance tradeoff**의 직관적인 예시이다.

### 이전 모델들과의 비교

| | Linear/Logistic/Softmax | kNN |
|---|---|---|
| 학습 | 파라미터(w, b) 학습 | 학습 없음 |
| 예측 속도 | 빠름 (행렬 곱) | 느림 (전체 데이터와 거리 계산) |
| 결정 경계 | 직선/평면 | 비선형 가능 |
| 메모리 | 파라미터만 저장 | 전체 학습 데이터 저장 |
