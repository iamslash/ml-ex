# Diffusion Model (확산 모델)

> 데이터에 노이즈를 점진적으로 추가한 뒤, 역으로 노이즈를 제거하여 데이터를 생성하는 모델.

## 실행 방법

```bash
cd pytorch-ex/diffusion
python diffusion.py
```

## 핵심 개념

### Forward Diffusion (노이즈 추가)

```
x_0 → x_1 → x_2 → ... → x_T (순수 노이즈)
```

각 스텝에서 조금씩 가우시안 노이즈를 추가. T 스텝 후 원본은 완전히 사라진다.

```
q(x_t | x_0) = N(sqrt(ᾱ_t) * x_0, (1 - ᾱ_t) * I)
```

### Reverse Diffusion (노이즈 제거 = 생성)

```
x_T (노이즈) → x_{T-1} → ... → x_0 (생성된 데이터)
```

신경망이 각 스텝에서 추가된 노이즈를 예측하고 제거한다.

### 학습 목표

```
Loss = MSE(실제 노이즈, 예측 노이즈)
```

신경망은 "이 noisy 데이터에 어떤 노이즈가 들어갔는지" 예측하는 법을 학습한다.

### VAE와의 비교

| | VAE | Diffusion |
|---|---|---|
| 잠재 공간 | 저차원 벡터 | 원본과 같은 차원 |
| 생성 과정 | 단일 디코딩 | T 스텝 반복 디노이징 |
| 학습 | ELBO 최대화 | 노이즈 예측 MSE |
| 생성 품질 | 흐릿한 경향 | 고품질 |
| 속도 | 빠름 | 느림 (T 스텝 필요) |

## 이 코드에서는

- 2D 나선형 데이터를 사용 (이미지 대신)
- T=100 스텝의 diffusion
- 생성된 점들의 분포가 원본과 유사한지 통계로 비교
