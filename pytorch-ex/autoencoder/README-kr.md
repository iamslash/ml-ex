# Autoencoder (오토인코더)

> 입력을 저차원으로 압축(인코딩)했다가 복원(디코딩)하는 비지도 학습 모델.

## 실행 방법

```bash
cd pytorch-ex/autoencoder
python autoencoder.py
```

## 핵심 개념

### 구조

```
입력 (784) → Encoder → 잠재 공간 (16) → Decoder → 복원 (784)
```

- **Encoder**: 고차원 → 저차원 (압축)
- **잠재 공간 (Latent Space)**: 데이터의 핵심 표현
- **Decoder**: 저차원 → 고차원 (복원)

### 학습 목표

입력과 출력이 같아지도록 학습 (자기 자신을 복원):

```
Loss = MSE(입력, 복원)
```

라벨이 필요 없다 → **비지도 학습**

### 왜 유용한가

- **차원 축소**: PCA의 비선형 확장
- **특징 추출**: 잠재 벡터가 데이터의 핵심 특징
- **이상 탐지**: 복원 오차가 큰 데이터 = 이상치
- **생성 모델의 기초**: VAE, Diffusion의 출발점

### VAE로의 확장

Autoencoder의 잠재 공간은 불규칙적 → 새로운 데이터 생성이 어렵다.
VAE는 잠재 공간을 정규분포로 강제하여 생성을 가능하게 한다.
